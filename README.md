# Cex Dex Arbitrage Repo
This repository contains an arbitrage project focused on identifying opportunities between __order book–based exchanges (CEX)__ and __AMM-based decentralized exchanges (DEX)__.
- Order books are primarily sourced from __Hyperliquid__
- AMMs are based on __Uniswap pools__

## Structure
- `prod/` — Production Code (Rust)
- `data_processing/` — Data collection and analysis (Python)
